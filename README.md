# STM

Stack-based virtual machine. See the README.md on each folder for more details.

## Exit codes

| Code | Meaning             |
|------|---------------------|
| 1    | Compilation failed  |
| 2    | Unreachable         |
| 3    | File not found      |
| 4    | Error parsing flags |
